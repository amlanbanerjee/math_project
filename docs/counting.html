3.1 Step Problem with Fibonacci Numbers

The number of ways to climb \(n\) stairs, taking either 1 or 2 stairs with each step, can be expressed as:

\(f(n) = f(n-1) + f(n-2)\)

where,

\(\forall n \in \mathbb{Z},\ n > 1\)

3.2 Race-with-Ties Problem

In this recursion, all possible scenarios of how a race can finish with any number of participants \(n\), can be expressed as:

\[ r(n) = \binom{n}{1}r(n-1) + \binom{n}{2}r(n-2) + \cdots + \binom{n}{n-1}r(1) + 1 \]

where,

\(\forall n \in \mathbb{Z},\ n > 1\)

\[ \binom{n}{m} = \dfrac{n!}{(n-m)! \cdot m!} \] represents the number of ways for \(m\) persons to be in 1st place, and

\(r(n-m)\) accounts for the number of arrangements of remaining participnats.
